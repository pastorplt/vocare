# Vocare Collective

Coming soon page for the Vocare Collective program — a community for discovering
your calling and living it out with purpose.

The site is a single static page (`index.html`) with no build step or
dependencies, ready to serve via GitHub Pages.

## Hosting with GitHub Pages

1. In the repo, go to **Settings → Pages**
2. Under **Build and deployment**, set Source to **Deploy from a branch**
3. Choose the branch and `/ (root)` folder, then save

The page will be published at `https://pastorplt.github.io/vocare/`.
